## 파일 삭제
1. 단일 파일 삭제
```bash
rm 파일명
```
2. 여러파일 삭제
```bash
rm 파일명1 파일명2 파일명3
```
3. 디렉토리와 그안의 모든 파일 삭제
```bash
rm -r 디렉토리명
```
4. 존재하지 않더라도 강제 삭제
```bash
rm -f 파일명
```
5. 확인 메세지 없이 디렉토리와 그 안의 파일 모두 삭제
```bash
rm -rf 디렉토리명
```
6. 확인 메세지 출력하고 파일 삭제
```bash
rm -i 파일명
```
7. 확인 메세지 출력하고 디렉토리 삭제
```bash
rm -ri 파일명
```
